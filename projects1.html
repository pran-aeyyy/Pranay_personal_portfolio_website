1. DermaLens (A Decentralized Dermatology Diagnostics App with Image Recognition)

-- DermaLens is an innovative app revolutionizing dermatological diagnostics.

-- Utilizing cutting-edge deep learning algorithms like InceptionV3, our platform empowers users to upload skin images for accurate disease identification.

-- Real-time Skin Analysis: Users can use their smartphone cameras to capture high-resolution images of their skin, moles, or other skin irregularities.

-- AI-Powered Diagnosis: The app employs deep learning models to provide instant, preliminary diagnoses.

-- Educational Resources: The app includes a blog section, where dermatology experts can share informative articles.

-- Doctor Collaboration: Users can opt to share their skin images and diagnostic results with healthcare professionals for remote consultations.





2. Video Analytics Project by Maharashtra Metro Rail Corporation Limited

Tech Stack- Python, OpenCV, PyTorch, YoloV7, CNN Architectures, AWS, Docker

-- This is the project proposed by Maharashtra Metro Corportaion for using Pre-installed cameras at metro stations for Autonomous Surveilance.

-- We worked on two problem statements:

  • Edge fall analytics : For edge fall analytics we had to monitor passenger movement near the platform edge in realtime and trigger an alarm in case of any accidents.

  • Ticketing queue analytics : We had to monitor the ticketing queues at the ticket counters and count the number of people. Using this we can track pattern of paasengers travelling from metro everyday.







    • 3. Automatic License Plate recognition using Yolov7 and Raspberry Pi 4B

    Tech Stack- Python, OpenCV, YoloV7, CNN Architectures, Roboflow, Computer Vision, Raspberry Pi 4B

    -- I chose to work on this project because i wanted to build a dataset of my own and build my model from scratch and train the model on my custom dataset.

    -- I built the entire dataset from scratch with Indian License plates. I collected around 600+ images and then augmented it which finally provided me with 1400+ images.

    -- Built the YoloV7 model from scratch. Split my dataset into 75% Training set, 15% validation set and 10% test set.

    -- The model was trained for 35 epochs and gave an accuracy of about 96%.
